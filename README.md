<div align="center">
   Okane is an easy expense and income tracker, which gives accurate graph representation of user's income/expenses/transactions. </br> This project consists of a website describing an app, and the app which enables users to track expenses/income/transactions and see their transactions visualization to plan their budget and secure their income.
</div>

<div align="center"><img src="https://okaneapp.netlify.app/okane_image.jpg" width="800"></img></div>

## 😀 User Can:

- [x] use the website to get all the available info about the app
- [x] contact okane team
- [x] see okane offices on the map
- [x] sign up/sign in/sign out
- [x] see the dashboard with the information about their income/expenses/transactions 
- [x] see graphs describing 'last 30 days' icome and expenses
- [x] see a list of transactions
- [x] add new transaction manually
- [x] see profile page

## 🧞 Instructions

To run the client:
```
open new terminal
npm install (first time only)
cd client
npm run dev
```
